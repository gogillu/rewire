# Rewire

Alternate endings of Bollywood movies, generated by 3 different LLMs.

- Backend: Go (TLS on :9999, modernc.org/sqlite — no CGO)
- Frontend: vanilla JS, Service Worker + IndexedDB
- LLMs: gpt-5.5, claude-opus-4.7, claude-sonnet-4.6 (via god backend)
- Public URL: https://gogillu.live:9999
